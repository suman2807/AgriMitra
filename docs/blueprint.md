# **App Name**: FarmLink

## Core Features:

- Crop Recommendation: Analyzes soil properties (NPK levels, moisture, temperature, rainfall) to suggest suitable crops.
- Fertilizer Suggestions: Provides fertilizer recommendations based on soil type, pH, temperature, and selected crop.
- Crop Disease Detection: Identifies crop diseases from uploaded images using an AI-powered image recognition tool.

## Style Guidelines:

- Primary color: Earthy green (#388E3C) to represent nature and growth.
- Secondary color: Light beige (#F5F5DC) to provide a neutral background.
- Accent: Golden yellow (#FFC107) for highlights and important information.
- Clean and organized layout with clear sections for each feature.
- Use relevant icons for soil properties, crops, and fertilizers.
- Subtle transitions and animations for a smooth user experience.

## Original User Request:
1. Crop Recommendation
FarmLink analyzes soil properties like nitrogen, phosphorus, potassium (NPK) levels, moisture, temperature, and rainfall to suggest the most suitable crops for your farm. This ensures optimized crop selection tailored to your unique environmental conditions.

2. Fertilizer Suggestions
Using data such as soil type, pH, temperature, and the selected crop, FarmLink provides precise fertilizer recommendations. These suggestions help maintain soil health, improve crop growth, and maximize overall yield efficiency.

3. Crop Disease Detection
With just an uploaded image of your crop, FarmLink AI-powered image recognition system identifies diseases and evaluates plant health. This allows for quick interventions to protect your crops and prevent widespread damage.
